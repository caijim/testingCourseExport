Manually Grade in-class or Offline Assignments

A fourth way in which you can use LTI components is to grade items that are not available in the course.

This can be very useful to online courses that either contain an in-person element or an assignment that cannot be submitted via the in-course materials.  

To enable this functionality, contact the CourseOps team.  We will then set up the appropriate configurations in your course.  This will involve creating a new unit in a subsection of your choosing, and creating an LTI component.  Once this LTI component is set up in your course, please:

  1. Mark the subsection the unit is in as a graded assignment.

  2. Update the number of assignments in your Grading page.



    You'll be left with a tool that can take a grade you (or CourseOps on your behalf) specify and display it in a participant's Progress page as well as the course grade report.
  3. Let us know when you've done this, and we will then create a new .csv file and send it to you.
  4. Once you have received and scored submissions for this in-class or otherwise offline assignment, refer to this file and input your participants' earned scores in the third column, followed by the maximum score possible, and any comments you want to add.  Note that the score earned cannot be higher than the maximum score possible.

  5. Save your changes and send this revised file back to us.
