If you want to add a file to your course, you can do so from the Files & Uploads page, available through the Content menu at the top of any Studio page. After you upload a file, you can add it to a course page with the "Embed URL" given on the Files & Uploads page. This is safer than pointing to a file hosted outside a course – sometimes external websites change their file or URL structures, and this will break any links pointing to old locations.

Go to the Files & Uploads page and copy the Embed URL of one of the images already there. Then come back to this page and edit this component. Click the little picture icon and paste in the Embed URL to add the image. Remember to Save and Publish your changes.
