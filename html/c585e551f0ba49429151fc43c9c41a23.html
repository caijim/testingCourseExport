The platform includes a wide variety of problem types that you can use to create assignments and assessments of all kinds. Click the problem component button at the bottom of this page to see a list of common problem types. Choose Multiple Choice, and an example problem with explanatory text will appear. Click Edit on the component to change the question and answer choices. Then click Settings to see the various options available for each question. 

Try adding an answer choice to the Checkbox problem below, then change the Maximum Attempts to 3 and make it so the Show Answer button will always be available. Also change the Display Name of the component to something descriptive, like Q1.1. Every component has a Display Name that students will see in the course.

Then try adding a problem of your own. You can explore the Advanced problem types too, to see what's possible in the platform.
